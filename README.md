Project Name: DigiTools Hub

Description:
DigiTools is a digital tools buying platform where users can explore, compare, and purchase useful tools for productivity,design,and development. It includes cart functionality and a smooth user experience.

Technologies Used:
React.js
Tailwind CSS
DaisyUI
JavaScript (ES6+)
React-Toastify

Features:
1.Add products to cart with live count update.
2.Remove items and clear cart using checkout.
3.Fully responsive design.
